Airfield Lighting Control and Monitoring Solution (AFL-CMS)

AFLCMS by UdangamiÂ® Technologie is an Airfield Lighting Control and Monitoring System (AFLCMS) allows the user to effectively control the various airfield lighting systems including runway, approach, taxiway, stop bars, apron etc. Software mimic panel that showcase everything clearly and allow control of field systems at fingertips.

Simple to use, the AFLCMS system meets a wide range of airfield needs and is ideal for small to medium airfields. The lights can be switched off and on individually or in groups. The reliable system is designed to be scaled as growing airports needs evolve.

AFLCMS can be easily adapted to interface to any type of airfield lighting equipment including a combination of equipment from different manufacturers. The control of several different airfield lighting systems can be integrated into a single homogeneous control panel.

  • Multiple Control Positions allowed
  • Single lamp control/monitoring
  • Radar Track Display on Airfield Layout
  • Runway Incursion Detection System
  • Alert on Flight Strip Occupancy
  • Alarm monitoring to Airfield Lighting Equipment
  • Event Data Logging System
  • Trend Analysis on Insulation Resistance to Field Installations
  • Interface with Flood Light Control System (Apron Management System)
  • Interface to Fixture Torque for Trend Analysis
